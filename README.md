# IskolE

IskolE is a role-based School Management System built using **PHP**, **MySQL**, and **Docker**. It is designed to streamline and simplify academic and administrative workflows within a school environment.

## 🚀 Features

- 🔐 Role-based access for:
  - Administrators
  - Teachers
  - Students
  - Parents

- 📊 Marks Management  
- 📅 Attendance Tracking  
- 🗓️ Timetable Scheduling  
- 📢 Announcements System  
- 📚 Academic Workflow Automation  

## 🛠️ Tech Stack

- **Backend:** PHP  
- **Database:** MySQL  
- **Containerization:** Docker  

## 🎯 Purpose

IskolE aims to provide a centralized platform that improves communication, enhances productivity, and ensures efficient management of school operations.
